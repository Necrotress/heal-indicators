# Heal Indicators+

<img src="https://raw.githubusercontent.com/Necrotress/heal-indicators/master/icon.png" width="64" height="64" align="right">

`Heal Indicators+` is a RuneLite plugin that shows heal splats whenever you heal with optional prayer restore splats, very similar to vitality plugin, with optional customization.

## Behavior

- Creates heal splats for positive HP gained.
- Creates prayer splats for positive prayer gained.
- Change colors of heal/prayer splats.

## Configuration

### General
- **Rise effect** — Toggle upward floating animation for splats
- **Position** — Show splats above the **Head** or **Body**
- **Splat duration** — How long splats remain visible (in game ticks)
- **Text outline** — Enable or disable black outline around numbers

### Heal & Prayer Splats
- **Enable heal/prayer splats** — Turn HP/Prayer heal indicators on/off
- **Show regeneration** — Show splats for passive healing and prayer regeneration
- **Splat color** — Custom color tint (for healing or prayer)
- **Splat size** — Scale of the splat icon (for healing or prayer)
- **Font size** — Size of the number displayed (for healing or prayer)
- **Test heal splat** — Spawn a random test heal splat
- **Test prayer splat** — Spawn a random test prayer splat
